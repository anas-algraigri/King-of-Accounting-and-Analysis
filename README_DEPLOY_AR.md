نسخة معدّلة من نظام مؤسسة عائش حميد القريقري للدواجن.

التعديلات الأخيرة:
- نقل القائمة الرئيسية إلى يمين الشاشة في العرض المكتبي والجوال.
- تفعيل إضافة وتعديل دليل الحسابات فعليًا مع حفظ التغييرات في التخزين المحلي.
- تفعيل الإضافة والتعديل من شجرة الحسابات، مع أزرار إضافة فرعي/تعديل/حذف.
- عند تغيير رقم حساب يتم ترحيل الرقم تلقائيًا في القيود، الفواتير، العملاء/الموردين، السندات، الرواتب، وكافة التقارير المرتبطة.
- تحسين ربط كشف الحساب، دفتر الأستاذ، ميزان المراجعة، وسجل الفواتير بفلاتر الحسابات والتواريخ.

تحديث v14.2:
- تحسين فواتير المبيعات والمشتريات بإضافة تاريخ التوريد، تاريخ الاستحقاق، UUID داخلي، حالة المستند، وحقول مرجع/سبب للإشعارات.
- إضافة ربط حساب لكل بند في الفاتورة من دليل الحسابات حتى تنعكس البنود على التقارير بدقة.
- تصحيح منطق الترحيل المحاسبي للفواتير والإشعارات الدائنة والمدينة للمبيعات والمشتريات.
- إضافة تعديل وطباعة/تصدير من سجل الفواتير، مع تحديث القيد المرتبط بدل إنشاء قيد مكرر عند التعديل.
- ربط تغييرات حسابات البنود مع ترحيل تغيير أرقام الحسابات في دليل الحسابات والشجرة.

تحديث v14.3:
- جعل قائمة الحساب في شاشة قيد اليومية تعرض الحسابات النشطة القابلة للترحيل من المستوى السادس فقط.
- منع حفظ القيد اليدوي إذا تم إدخال حساب من مستوى أعلى أو أدنى من المستوى السادس.
- استمرار ربط القيود المحفوظة بالتقارير وكشف الحساب وميزان المراجعة بدون فصل أو تكرار.

طريقة الرفع:
ارفع كامل الملفات كما هي إلى GitHub أو Vercel. نقطة التشغيل هي app/index.html، ويمكن التشغيل محليًا بالأمر:

npm start


تحديث v14.5:
- جعل كل قيد يومية يدوي أو مولد من الفواتير يستخدم حسابات المستوى السادس فقط، مع عرض الحساب الرئيسي بجانب الحساب مثل العملاء/الموردون/المبيعات/المشتريات.
- ترحيل فواتير المبيعات والمشتريات والإشعارات الدائنة والمدينة على حسابات المستوى السادس، مع إبقاء اسم العميل/المورد في البيان والسجل.
- إضافة مقارنة مباشرة لضريبة القيمة المضافة بين الفواتير والقيود، وزر لتحديث ربط الفواتير والإشعارات بالقيود والتقارير.
- إضافة حاسبة الوعاء الزكوي والزكاة بنسبة افتراضية 2.5% مرتبطة بأرصدة حسابات المستوى السادس وبصافي الربح المحاسبي.
- تحسين هوية ملفات Excel و PDF المصدرة بإضافة رأس رسمي، بيانات المنشأة، التاريخ، علامة الميزان، وتنسيق الجداول حسب هوية النظام.
