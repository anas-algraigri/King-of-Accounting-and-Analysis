# تحديث صلاحيات المستخدمين v15.10

تم تطبيق الصلاحيات فعليًا على جميع صفحات النظام والقائمة والبحث السريع والاختصارات.

- إذا لم يحصل المستخدم على صلاحية صفحة، لا تظهر في القائمة.
- إذا حاول فتحها يدويًا، يتم منعه برسالة تنبيه.
- الصلاحيات تُحفظ حسب المربعات المحددة حتى لو كان الدور مدير/محاسب/مبيعات.
- لا تغيير في القيود أو الفواتير أو التقارير أو الربط السحابي.
