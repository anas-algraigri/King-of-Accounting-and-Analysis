# King Accounting v15.13 Performance Safe

تحديث تجربة التسريع بدون حذف المرفقات.

## التعديلات
- إلغاء أي خيار يحذف أو يقلل المرفقات.
- المرفقات محفوظة داخل حالة النظام وتنتقل مع النسخ الاحتياطية والسحابة.
- تفعيل الوضع السريع فعليًا: يتم تحديث الصفحة المفتوحة فقط بدل إعادة تحميل كل الصفحات.
- زر تحديث الصفحة الحالية.
- زر تحديث كل الصفحات عند الحاجة.
- ترقيم صفحات تلقائي للجداول الكبيرة.
- تأخير الحفظ السحابي التلقائي قليلًا لتخفيف التجميد عند وجود مرفقات.
- تحسين تصدير Excel/PDF بإظهار انتظار قصير وعدم تجميد الواجهة قدر الإمكان.

## ملاحظة
لم يتم تغيير آلية القيود أو الفواتير أو الرواتب أو الحسابات أو الربط السحابي.
