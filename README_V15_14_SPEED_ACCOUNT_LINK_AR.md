# تحديث v15.14 - تسريع وربط حسابات الفواتير

تم بناء هذه النسخة على v15.13 Performance Safe.

## التحديثات
- تحسين إضافي للأداء عبر فهرسة الحسابات والحركات وتقليل التحديثات العامة.
- إبقاء المرفقات محفوظة وعدم حذفها.
- تحسين بحث كشف الحساب ودفتر الأستاذ: الاسم لا يعود بعد حذفه، والرقم يظهر فقط عند اختيار/تطابق واضح.
- ربط حسابات الفواتير بالدليل والشجرة: اسم الحساب في خانة ورقم الحساب في خانة ثابتة.
- تحسين عرض حساب العميل/المورد في الفواتير برقم الحساب بصيغة النظام.
- لم يتم تغيير آلية القيود أو الفواتير أو الرواتب أو الربط السحابي.
