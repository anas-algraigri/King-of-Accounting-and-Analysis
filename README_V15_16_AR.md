# تحديث v15.17 - ميزان مراجعة احترافي وحساب ضريبة موحد

تم بناء هذا التحديث على نسخة v15.16 مع تعديلات محدودة وآمنة بدون تغيير آلية الفواتير أو القيود أو الرواتب أو المرفقات أو الربط السحابي.

## التحديثات المنفذة
- تحسين ميزان المراجعة ليعمل بتجميع أسرع للحركات بدل فحص كل حساب مع كل سطر.
- جعل البحث في ميزان المراجعة وتحديد المستوى أكثر سلاسة، مع دعم عرض المستوى المختار للحسابات الرئيسية والفرعية.
- إضافة ملخص احترافي في ميزان المراجعة يوضح النطاق، عدد الصفوف، إجمالي الحركة، وفرق التوازن.
- توحيد ضريبة المخرجات وضريبة المدخلات في حساب واحد باسم: **هيئة الزكاة والضريبة والجمارك**.
- جعل حساب الضريبة الموحد ضمن **الدائنين / الالتزامات المتداولة**.
- ترحيل ضريبة المخرجات كحركة دائنة على حساب الهيئة، وضريبة المدخلات كحركة مدينة على نفس الحساب، ليظهر الصافي في التقارير.
- ربط حساب الهيئة الموحد بتقارير الضريبة، مؤشر الترابط، القيود، والفواتير بدون تغيير المعادلات الأساسية.
- إضافة ترحيل آمن للحركات القديمة من حساب ضريبة المدخلات السابق إلى حساب الهيئة الموحد عند فتح النظام.
- تحسين سرعة النظام في التقارير الثقيلة عبر تجميع ميزان المراجعة مرة واحدة، والاستفادة من فهارس الحسابات والحركات الموجودة.
