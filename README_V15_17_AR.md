# تحديث v15.17 - توحيد حسابات العملاء والموردين مستوى سادس

تم بناء هذا التحديث على نسخة v15.16.

## التحديثات
- أي عميل أو مورد يتم فتحه من الفواتير أو العملاء والموردين يتم إنشاء حسابه كمستوى سادس تحت بند العملاء أو الموردين.
- حسابات العملاء والموردين تظهر في دليل الحسابات وشجرة الحسابات وبحث الحساب في القيود.
- عند اختيار عميل أو مورد في الفاتورة يتم ربط الحساب المرتبط مباشرة من الدليل والشجرة.
- فواتير الآجل تستخدم حساب العميل أو المورد المرتبط بدل الحساب العام.
- أي تعديل في اسم/رقم حساب العميل أو المورد يتم توحيده مع الفواتير والقيود والتقارير.
- الإبقاء على آلية القيود والفواتير والرواتب والمرفقات والربط السحابي بدون تغيير.
