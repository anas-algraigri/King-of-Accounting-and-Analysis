# King Accounting v15.1

تحديث عرض تواريخ التقارير والتصدير:

- إظهار تاريخ القيد المختار من خانة التاريخ داخل PDF وExcel للقيد الحالي.
- إضافة تاريخ القيد كعمود داخل تصدير القيد.
- إظهار فترة/تاريخ التقرير في رأس PDF وExcel لجميع التقارير المهمة: سجل القيود، الفواتير، كشف الحساب، ميزان المراجعة، القوائم المالية، الضريبة والزكاة، وإقفال السنة.
- تسمية ملفات Excel بتاريخ التقرير المختار عند توفره.
- الحفاظ على الربط السحابي، المرفقات، إقفال السنة، الرواتب، والفواتير بدون تغيير آلية القيد السابقة.

## الرفع
ارفع `index.html` في المكانين `/index.html` و`/app/index.html`. وارفع `supabase-config.js` في المكانين إذا احتجت.
