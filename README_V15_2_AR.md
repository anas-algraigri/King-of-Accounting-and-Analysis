# King Accounting and Analysis v15.2

## التحديثات
- ترقيم القيود أصبح آليًا ومتسلسلًا بدون تكرار مع جميع الإجراءات التي تولد قيودًا، مثل الفواتير والسندات ومسيرات الرواتب والإقفال.
- رقم القيد في شاشة القيد أصبح ثابتًا/مقروءًا فقط، ويتم توليده آليًا.
- زر حفظ القيد لا يكون فعالًا إلا بعد وجود تغيير، وعند تعديل قيد موجود يتم تحديث نفس القيد دون إنشاء قيد مكرر.
- حذف القيد أصبح حذفًا فعليًا من السجل والتقارير بدل وضعه في حالة محذوف.
- إضافة زر إعادة ترقيم القيود، مع تحديث مراجع الفواتير والسندات ومسيرات الرواتب بعد إعادة الترقيم.
- تحديث مسير الرواتب بإضافة خانة المكافآت.
- إضافة توجيه محاسبي لكل بند في مسير الرواتب: الراتب الأساسي، البدلات، المكافآت، والاستقطاعات.
- تحديد الحساب المدفوع منه الراتب بخيارات مرتبطة: الصندوق، البنك، جاري المالك، العهدة.
- حذف مسير الرواتب يحذف قيده المرتبط فعليًا من التقارير.
- الحفاظ على الربط السحابي، المرفقات، إقفال السنة، القوائم المالية، الفواتير، وميزان المراجعة كما في الإصدارات السابقة.
