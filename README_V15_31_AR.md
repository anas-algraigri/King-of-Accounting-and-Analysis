# v15.31 - تحسين الشعار والرأس والتذييل

تم في هذه النسخة:
- تحسين عرض الشعار داخل رأس الفاتورة مع قصه وتنظيفه بصريًا.
- إعادة تصميم رأس الفاتورة بشكل احترافي مع اسم المؤسسة بالعربي والإنجليزي وشارة للرقم الضريبي.
- إعادة تصميم التذييل بشكل أنظف وأكثر رسمية مع معلومات المؤسسة بالعربية والإنجليزية.
- الإبقاء على QR الضريبي الأبيض، ترقيم INV، وربط النظام كما هو.
- عدم التأثير على القيود أو الضريبة أو التقارير.

طريقة الاستخدام:
- افتح ملف HTML في المتصفح.
- أنشئ أو افتح الفاتورة.
- اطبع الفاتورة أو صدرها PDF، وسيظهر الشكل الجديد للرأس والتذييل.
