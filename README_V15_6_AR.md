# تحديث v15.6 - ربط العملاء والموردين بالفواتير والقيود

## التحديثات
- في فاتورة المبيعات يمكن إدخال اسم العميل يدويًا أو اختيار عميل موجود من قبل.
- في فاتورة المشتريات يمكن إدخال اسم المورد يدويًا أو اختيار مورد موجود من قبل.
- عند اختيار سجل موجود يتم جلب الرقم الضريبي والسجل التجاري والعنوان والحساب المرتبط.
- منع إنشاء عميل أو مورد مكرر عند تطابق الاسم أو الرقم الضريبي أو السجل التجاري.
- ربط سجلات العملاء والموردين بدليل الحسابات وشجرة الحسابات والفواتير والقيود والتقارير.
- زر تحديث وربط شامل يقوم بمراجعة ودمج السجلات المكررة وإعادة ربط الفواتير بقيودها.
