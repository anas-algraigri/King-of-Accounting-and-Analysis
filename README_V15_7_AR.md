# تحديث v15.7 - الحذف الآمن وإدخال عميل/مورد جديد مستقل

## التحديثات
- إضافة حذف نهائي للعميل أو المورد من صفحة العملاء والموردين.
- قبل الحذف يتم فحص الفواتير والإشعارات والقيود والحساب المرتبط.
- إذا كان العميل أو المورد مستخدمًا تظهر رسالة تفصيلية بأرقام الفواتير وأرقام القيود المرتبطة ولا يتم الحذف حتى لا يحدث خلل.
- تفعيل الحذف الفعلي في دليل الحسابات وشجرة الحسابات مع تنبيه يوضح أرقام القيود أو الفواتير أو السندات أو المسيرات المرتبطة بالحساب.
- تفعيل زر إدخال جديد في فواتير المبيعات والمشتريات بحيث لا يجبر النظام على اختيار اسم مشابه من العملاء أو الموردين.
- عند إدخال جديد مستقل يتم إنشاء سجل جديد إذا لم يوجد تطابق تام بالاسم أو الرقم الضريبي أو السجل التجاري.
- الحفاظ على الربط مع دليل الحسابات، شجرة الحسابات، القيود، الفواتير، التقارير، الرواتب، والربط السحابي.
