# تحديث v15.8 - حذف العميل/المورد مع تأكيد نهائي

## الجديد
- يمكن حذف العميل أو المورد نهائيًا من صفحة العملاء والموردين حتى لو كان مرتبطًا بفواتير أو قيود.
- قبل الحذف يظهر تنبيه يوضح أرقام الفواتير/الإشعارات والقيود والحسابات المرتبطة.
- بعد التنبيه تظهر رسالة تأكيد نهائية قبل تنفيذ الحذف.
- لا يتم حذف الفواتير أو القيود التاريخية؛ تبقى محفوظة بأرقامها وتظهر في التقارير.
- إذا كان حساب العميل/المورد غير مستخدم يتم حذفه من دليل الحسابات والشجرة، وإذا كان مستخدمًا يبقى لحماية التقارير والقيود.

## طريقة الرفع
ارفع ملف `index.html` في المكانين:

- `/index.html`
- `/app/index.html`

وارفع `supabase-config.js` في المكانين إذا احتجت تحديث بيانات الربط:

- `/supabase-config.js`
- `/app/supabase-config.js`
