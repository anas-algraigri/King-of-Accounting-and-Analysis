# تحديث v16.1 — فصل قيود السنوات المالية

## ما الجديد
- إضافة صفحة مستقلة باسم **قيود السنوات المالية**.
- كل قيد يتم إنشاؤه من أي عملية في النظام يظهر تلقائيًا في صفحة السنة المالية حسب تاريخ القيد.
- يشمل ذلك:
  - القيود اليدوية
  - فواتير المبيعات والمشتريات والإشعارات
  - سندات القبض والصرف
  - مسيرات الرواتب
  - قيود إقفال السنة
  - القيود الافتتاحية للسنة الجديدة
- إضافة بحث مرن داخل صفحة قيود السنوات المالية برقم القيد، الاسم، البيان، الحساب، نوع القيد، أو مصدر العملية.
- إضافة تصدير Excel و PDF لقيود السنة المالية.
- لم يتم تغيير آلية القيد أو المعادلات أو الربط السحابي أو كل تحديثات v16.0 السابقة.

## طريقة الرفع
ارفع ملف index.html في المكانين:
- /index.html
- /app/index.html

واحتفظ بملف supabase-config.js كما هو في المكانين إذا كان موجودًا.
