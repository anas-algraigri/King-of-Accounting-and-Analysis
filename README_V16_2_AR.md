# تحديث v16.2 - البحث الاحترافي والترابط الشامل

## التحديثات
- توحيد وتحسين البحث المرن في جميع التقارير والعمليات.
- كشف الحساب: بحث كتابي احترافي باسم الحساب أو رقم الحساب أو جزء من الاسم.
- دفتر الأستاذ: بحث مرن باسم/رقم الحساب.
- ميزان المراجعة: خيار جميع الحسابات أو حساب/مستوى محدد، مع البحث بأي مستوى مثل الأصول أو البنك.
- شجرة الحسابات: إظهار الرصيد والحركة بجانب كل حساب مع بحث مرن في جميع المستويات.
- دليل الحسابات: ترتيب أفضل، وفرز من أعلى الصفحة حسب الرقم أو الاسم أو النوع أو المستوى أو الرصيد أو الحركة.
- تعزيز تحديث الترابط بعد التعديل أو الحذف للفواتير والسندات والرواتب والقيود.
- الحفاظ على كل تحديثات النسخ السابقة: الربط السحابي، قيود السنوات المالية، الترقيم، الحذف، المرفقات، وإقفال السنة.

## طريقة الرفع
ارفع ملف index.html في المكانين:
- /index.html
- /app/index.html

وارفع supabase-config.js إذا لم يكن موجودًا أو إذا أردت تحديث بيانات Supabase.
