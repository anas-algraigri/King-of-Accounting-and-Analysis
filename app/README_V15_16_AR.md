# تحديث v15.16 - ربط العميل/المورد وحسابه في الفواتير مباشرة

تم بناء هذا التحديث على نسخة v15.15، مع تعديل محدود على الفواتير فقط.

## التحديثات
- تحسين خانة اختيار العميل/المورد الموجود من قبل في فواتير المبيعات والمشتريات والإشعارات.
- عند اختيار العميل/المورد يتم جلب بياناته مباشرة:
  - الاسم
  - الرقم الضريبي
  - السجل التجاري
  - العنوان
  - رقم/اسم الحساب المرتبط من دليل الحسابات وشجرة الحسابات
- خانة رقم/حساب مرتبط أصبحت تتحدث مباشرة عند الاختيار أو عند وجود تطابق كامل.
- لا يتم إجبار المستخدم على اسم مشابه عند إدخال عميل/مورد جديد.
- عند حفظ الفاتورة يتم توحيد الربط بين العميل/المورد والدليل والشجرة والقيود والتقارير.
- لم يتم تغيير آلية القيود أو الفواتير أو الرواتب أو التقارير أو الربط السحابي.
