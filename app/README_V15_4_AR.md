# تحديث v15.4 - توجيه مسير الرواتب بحساب دفع واحد

## أهم التحديثات
- تعديل شاشة مسير الرواتب بحيث يتم اختيار **حساب دفع واحد فقط**.
- حساب الدفع محصور في: الصندوق، البنك، جاري المالك، العهدة.
- إلغاء خانات توجيه كل بند يدويًا من شاشة المسير لمنع الخطأ.
- التوجيه المحاسبي للبنود أصبح آليًا:
  - الراتب الأساسي → مصروف رواتب وأجور.
  - البدلات → مصروف بدلات ومزايا.
  - المكافآت → مصروف مكافآت موظفين.
  - الاستقطاعات → مصروفات/التزامات رواتب مستحقة.
  - صافي الراتب → دائن على حساب الدفع المختار.
- حفظ وتعديل المسير يحدث نفس القيد المرتبط ولا ينشئ قيدًا مكررًا.
- حذف المسير يحذف قيده المرتبط فعليًا من التقارير.
- تحسين مؤشر الترابط ليشمل صحة توجيه مسير الرواتب.
